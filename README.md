# 📊 Superstore Sales Dashboard

An interactive Excel Dashboard built using the Superstore Sales Dataset to analyze sales performance across different business dimensions. This project demonstrates data cleaning, pivot table analysis, dashboard design, and data visualization using Microsoft Excel.


## 📌 Project Overview

The objective of this project is to transform raw sales data into an interactive dashboard that provides meaningful business insights. The dashboard allows users to analyze sales performance across different regions, categories, customer segments, shipping modes, and top-performing states.

This project showcases practical Excel skills commonly used in Data Analytics roles.


## 🎯 Objectives

- Clean and prepare raw sales data
- Perform sales analysis using Pivot Tables
- Build an interactive dashboard
- Visualize business performance using charts
- Enable filtering using slicers
- Present insights in a clear and professional format


## 🛠 Tools Used

- Microsoft Excel
- Pivot Tables
- Pivot Charts
- Slicers
- Conditional Formatting
- Data Cleaning
- Data Visualization


## 📈 Dashboard Features

### 📍 Sales by Region
Displays total sales generated from each region.

### 🛒 Sales by Category
Compares sales across Furniture, Office Supplies, and Technology.

### 👥 Sales by Segment
Analyzes sales generated from Consumer, Corporate, and Home Office customers.

### 🚚 Sales by Ship Mode
Shows sales distribution based on shipping methods.

### 🏆 Top 10 States by Sales
Highlights the highest-performing states based on total sales.

### 🎛 Interactive Slicers
Users can filter the dashboard dynamically by:

- Region
- Category
- Segment
- Ship Mode
- State


## 📊 Key Insights

- Technology generated the highest sales among all product categories.
- Standard Class contributed the highest sales among shipping modes.
- Consumer Segment recorded the highest sales.
- California emerged as the top-performing state.
- West Region generated the maximum sales.
